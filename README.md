A lightweight web application built with Python Flask that evaluates the strength of user passwords based on commonly accepted security criteria. The application classifies passwords as Weak, Medium, or Strong, helping users understand and improve password security.

**Overview**
Password security is a critical aspect of modern applications. This project demonstrates a simple yet effective approach to validating password strength using rule-based checks implemented with Python and regular expressions.
The application provides a clean user interface and processes password input securely on the server side.

**Features**
1.Server-side password strength evaluation
2.Rule-based scoring mechanism
3.Classification into Weak, Medium, and Strong categories
4.Simple and responsive user interface
5.Beginner-friendly Flask architecture

**Technologies Used**
1.Programming Language: Python
2.Web Framework: Flask
3.Frontend: HTML, CSS
